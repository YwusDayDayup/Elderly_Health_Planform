# 社区养老系统 — 接口契约文档

> 用途：UI 重构期间保证前端接口调用不变，防止功能回归。
> 所有接口均通过 `frontend/src/utils/request.ts`（axios 实例）发起，统一响应格式见下方。

---

## 一、统一响应格式

```typescript
interface Result<T = any> {
  success: boolean   // true = 成功
  code: string       // 业务码，成功为 "0" 或 "200"
  message: string    // 提示信息
  data: T            // 业务数据
}
```

**分页结果：**
```typescript
interface PageResult<T> {
  total: number   // 总条数
  list: T[]       // 当前页数据
}
```

## 十八、重构安全规则

1. **不修改任何 API 函数签名**（函数名、参数类型、返回类型）
2. **不修改接口路径**（URL 字符串）
3. **不修改请求方法**（GET/POST/PUT/DELETE/PATCH）
